# 📊 Excel Sales Dashboard

## 📌 Project Overview

This project showcases an interactive Sales Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, and business trends. The dashboard enables users to monitor key performance indicators (KPIs), identify top-performing products, and make data-driven business decisions.

---

## 🎯 Business Objective

The goal of this dashboard is to help management answer questions such as:

- What is the total sales revenue?
- Which products generate the highest sales?
- Which region performs the best?
- How do monthly sales trends change over time?
- Which customer segments contribute the most revenue?

---

## 📁 Dataset

The dataset contains transactional sales data including:

- Order ID
- Order Date
- Customer Name
- Product
- Category
- Region
- Sales
- Quantity
- Profit

*Note: This dataset is for learning and portfolio purposes.*
